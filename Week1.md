# Week 1 Notes

## Why Learn Java
- Java is one of the most popular programming languages.
- Java is used for object-oriented programming (OOP).
- Programming is about solving problems, not just writing code.

## Object-Oriented Programming (OOP)
The four pillars of OOP:
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## Encapsulation
- Protect data by making fields private.
- Use getters and setters to access data.

## Access Modifiers
- public = everyone can access
- private = only the class can access
- protected = class, package, and subclasses
- package-private = package only

## UML
- UML is a blueprint for a class.
- + means public.
- - means private.
- UML shows:
  - Class name
  - Fields
  - Constructors
  - Methods

## Unit Testing
- JUnit checks if code works correctly.
- @Test marks a test method.
- assertEquals() compares expected and actual values.

## Git
Important commands:
- git add
- git commit
- git checkout -b
- git push

Git is used to save changes and manage versions of code.

## Generics
A generic is a container.

Examples:
- List
- ArrayList
- HashMap

Important methods:

List
- add()
- size()

HashMap
- put()
- get()
- keySet()
- values()
- entrySet()

## Things to Remember
- Class = Blueprint
- Object = Instance of a class
- Getter = Gets data
- Setter = Changes data
- Constructor = Creates an object
- Unit tests check if code works.
